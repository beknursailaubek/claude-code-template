# Default Output Style

- Lead with the answer or action, not reasoning
- Max 3 sentences per explanation unless asked for more
- Bullet lists for multi-point answers
- No emojis, no repetition of user's words
- Show code diffs, not descriptions of changes
- If uncertain, state it in one sentence and propose the smallest safe change
